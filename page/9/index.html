今日も天気が良くあたたかいが、三日前に買ってきた富有柿はまだ好みの硬さにまで熟していない。食べごろはもう少し先になりそうだ。朝食をさくっと済ませてからぬか床をかき混ぜる。ぬか床からぬかっぽい匂いがするけど、健全なぬか床はどんな匂いがするんだろう…、ぬか漬けの達人がいたら嗅がせてほしい。

仕事はゆっくりと進めることができた。リリース作業に加えて各種案件に関する調整やいくつかの PR を出し、その後は Elasticsearch についての学習を進めていた。refresh が内部（Lucene）で行っている処理や filter 、Node query cache について概ね理解できた気がするが、ただ filter がどういう単位でキャッシュされるのかがよくわかっていない。

仕事を終えてぬか床をかき混ぜる。一度本漬けしてみようということできゅうりを漬けてぬか床ごと冷蔵庫に入れた。冷蔵庫の場合は 24 時間が目安とのことなので、明日の夕食で食べてみる予定だ。食べてみて、仕上がりによっては捨て漬けに戻ったり塩を足したりといった判断をしなければならない。ぬか床は生き物だからちゃんと愛情をかける必要があるというのをどこかで読んだけれど、少しずつ意味がわかってきた気がする。

ずっと欲しかった山田工業所の 24cm の揚げ鍋を注文した。これまでテフロン加工されたフライパンで揚げていたので、フライパンによくないだろうなという気持ちがあった。揚げ鍋は鉄製なのでしっかり手入れすればガシガシと普段遣いできると思う。届くのが楽しみだ。

📝 朝食

・玉子焼きを沈めた昨日の味噌汁
・かぼちゃの煮つけ（あまりもん）

📝 昼食

・チャーハン

しょうがとにんにく、青ねぎをたくさん使ったチャーハン。油に直接塩を撒いたほうが味が均等になり美味しく感じる。焦がした薄口醤油の香りを移してあっさりと仕上げた。

📝 夕食

・ごはん
・ぶりの照り焼き（長ねぎを添えた）
・青梗菜と厚揚げの煮浸し
・かぼちゃの煮つけ（あまりもん）
・さつまいもと長ねぎとベーコンの味噌汁

青梗菜はごま油で軽く炒めてから煮浸しにした。ぶりの照り焼きはいつ食べても美味しい。
