PHP 8.0.0 をインストールしてから変更点の確認や最近のツールチェインを調べていて 2 時に寝た。そして今日は歯医者だったことを思い出し 8:30 に起きる。急いで支度をして歯医者に向かったが、予約は 9:30 とのことでまるまる 30 分待つことに。もう少し寝ておけばよかった。次は半年後に検診に来てくださいとのことだった。

昼前に幼なじみがこっちまで来るというので、さくっと朝食と朝食を済ませる。11:30 に合流してそのまま焼肉屋にいった。久しぶりの牛肉、美味しかった。それから先週通りがかって気になっていた近所の喫茶店へ。これが予想外にむちゃくちゃ雰囲気がよい。alt 地元の喫茶店だ。閉店時間が 21 時と早いのがあれだけど、おそらくこれから定期的に通うことになると思う。雨が降っていたのでスーパーと薬局にも車で連れてってもらって 14:30 に別れた。

さー、趣味開発を進めるぞという気持ちだったが今日のうちに換気扇と魚焼きグリルの大掃除をしたいとも思っていたので先にそちらを進める。お湯に重曹を溶かしつけ置きしたが、じきに茶色いほこりや汚れが浮いてきて換気扇も定期的に掃除しないといけないなという気持ちになる。思えば去年の年末から一度も洗っていなかった。

つけ置きいている間に引き続きツールチェインを調べる。はじめて bamarni/composer-bin-plugin を知って感動した。linter や static analysis に関するツールチェインは決してアプリケーションの依存コードではないのに、dev dependencies に書くことに違和感を持っていたからだ。そうした違和感をこのパッケージは解消してくれる。

換気扇はつけ置きの効果があって洗えばすぐに汚れを落とすことができた。しかし魚焼きグリルはいにしえの焦げ付きをどうしても剥がすことができなかった…、これはなにかで頑張るよりは網の部分だけ部品を買ったほうがよいのではないかという気持ちになる。

気づけば時刻は 20 時で、何もできてないとつらくなる。お米を仕掛けつつ、自分のために PHP パッケージの開発に使えるスケルトン riverside-jp/php-package-skeleton をこの機会に作ったので公開した。最小限の構成になっているので、依存パッケージのアップデートだけしっかりしていけば今後も使えると思っている。

夕飯と片付けを済ませたらすでに 22 時で、何もできないまま一日が終わろうとしている…。寝不足であまり頭も回ってないしなかなかうまくいかないものだなと思う。久しぶりの趣味開発は毎日少しずつ進めていくことにしよう。
