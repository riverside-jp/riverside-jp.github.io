ポッキーの日らしい今日、コロナの感染者数は相変わらず右肩上がりとなっており、再来週に都道府県を越えての移動を伴う予定がある自分にとっては、その予定をどうにかできないものかと少し憂いている（どうにかできないタイプの予定なのでどうしようもないけれど）。

天気が特別に良かったから、朝からシーツを洗って布団を干した。敷きシーツを敷毛布に変え、掛毛布を出して衣替えをした。11 月とはいえまだまだ暖かく、このままずっと半袖で過ごせるのではないかと思ってしまう。しかしスーパーには冬の野菜や果物が並び始めていて、少しずつ冬が近づいているのは確からしい。

Elasticsearch 難しい…、Primary Shard と Replica Shard の数をどのように決めるべきかをよく理解していない。Write より大幅に Read の割合が大きい用途だが、その場合に Replica Shard を際限なく増やしても良いのかなどの疑問がある。おそらく Replica Shard の分だけインデックスのパフォーマンスは落ちるが、その影響が許容範囲内かどうかなどはちゃんと検証して判断するしかないのだろう。泥臭くやっていくしかない。
