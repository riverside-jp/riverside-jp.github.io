連日良い天気だったが、今日は朝から曇り。掃除と朝食を済ませて半袖でセブンコーヒーを買いに行ったが、陽が照ってないのでさすがに少し寒い。

今週は少しばたばたしていたので、仕事はセーブモード。今日中にやらなければならないことはないので気ままに進めさせてもらう。合間に GitHub 用のアイコンをまともなものにしたいと思っていたのでサクッと作り直した。32px * 32 px だと視認性が悪いのでどうにかした方がよいかもしれない。

お昼はぬか漬けを取り出して、ご飯と味噌汁で食べた。この組み合わせをやりたいと思っていたからとても嬉しい、そしてむちゃくちゃおいしかった。ごぼうは少しあくを感じたから下ごしらえをもっと工夫しないといけないのかもしれない。

夜は早めに切り上げて麻婆豆腐を作った。意図的に挽肉と赤味噌を焦がすようにすればなお旨味が強くなる。それはそうと長ねぎを蛇腹切りできないくらいには包丁が切れなくなってるのでいい加減研がないといけない…。

📝 夕食

・ごはん
・麻婆豆腐（土井善晴先生のレシピ）
・ほうれん草のにんにく炒め
・たまごと豆腐のお吸い物
・にんじんと大根のぬか漬け

土居善晴先生曰く、野菜の炒めものは炒めたあとのかさが少なくなったところに自分ならどれくらいの塩を振るかを考えて振れば良いとのこと。実際にほうれん草の炒めものには小さじ 1/5 程度しか塩を入れていないが、十分な塩加減に感じた。1/6 でも良いくらいだ。

たまごのお吸い物を作るときはいつもだしに対して 8% の薄口醤油を入れるが、全体のバランスを考えて今日は 5% にしてみた。大正解だった。これ単品では少しぼやっとした味つけだが、全体を構成するひとつとして捉えるとほっとする味つけになっていた。塩分のコントロールが少しずつうまくなってきている実感がある。
