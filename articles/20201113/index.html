昨日とうってかわって今日はあたたかい。朝食を食べてからぬか床を底からかき混ぜる。知らないうちに中指の先を切っていて染みる…、明日の晩には捨て漬けの野菜を入れ替えないといけそうだ。

Redis から特定のキーだけデータが消えるという現象が起きていて、当初はその特定のキーに関連する部分のアプリケーションの変更をしたためそれが影響しているのか・・？しかし関係ないはずだけどな、と頭を抱えていたが、単純に Redis のメモリ使用量が上限に達しているだけであった。 maxmemory-policy は volatile-lru な環境のため、その作成したての特定のキーが削除対象に含まれる理屈がよくわからない。いずれ検証したいと思う。

夜に日本武道館で行われるライブに行く予定があったため、16 時には仕事を終えて駅へ。

会場に向かう電車はかなり空いていたが、第三波が来ていることもあり何をするにも緊張感がある。収容を 50% 以下に抑えているということで、会場につくも人はまばらで、いつもよりもひっそりとしていた。グッズ販売は中道場という建物のなかで行われていて、決められた導線上しか移動できないようになっていた。これが新様式か・・！

入場して席につくも、アリーナはブロックごとの間隔が大きく取られて、席もひとつ飛ばしに整備されていた。自分は早い時間帯に分散入場したため、開演まで 1 時間半ほど待ちが発生したのだが、換気が強いのか冷気がすごい。運営は考えられる範囲での感染対策をしっかりとしてくれていたが、しかしながら皆さん結構会話などをしており、難しいところがあるよなあと思う。

公演はとても素晴らしいものだった。歓声も禁止、歌声も禁止で、観客ができることは拍手くらいなのだが、普段はもっと賑やかな分これはこれで新鮮だった。とはいえ、また皆で歌えるような世の中が戻ってきてほしいと思う。

自分は間接的にエンターテイメントを届ける仕事をしているが、エンターテイメントは人々に必要なのか、自分の仕事は世の中の役に立っているのかと悩むことがしばしばある。しかし、このご時世に自分自身でエンターテイメントを享受したことで、やっぱりエンターテイメントは必要なんじゃないかと思った。生命の維持には必要ないかもしれないが、人間として生きていくためにはきっと必要だ。そう考えると、自分の仕事も少しは世の役に立っているのかもしれない。

公演が終わり、電車で錦糸町に移動してざっくりと調べて出てきたラーメン屋に入った。これが美味しいものではなかった…。家ごはんが恋しい。

・豚汁（あまりもん）

📝 昼食

・ペペロンチーノ

アルミパンを使うようになってから明らかにパスタの仕上がりが変わった。にんにくへの火の入れ具合がコントロールしやすくなったのが大きいと思う。

📝 夕食

・醤油ラーメン 味玉付き

おそらく鶏となんらかの白身魚がベースのスープだったが、完成度がいまいちであった…。スープから魚の皮目のにおいを感じてしまい、残念な気持ちに。スープの軽さに対して、麺が太く加水率も高めで馴染んでいない感が否めない。食べログ 3.6 超えだったので立ち寄ったが、札幌でやっていくとしたら難しいだろうなという完成度だった。
